自慰器推销员

  • 发表于： 8月 3, 2018 8月 3, 2018
  • 分类： 大福音書
  • 标签： 巨根崇拜

「 叮噹！」門鈴響起來。

「 是，來了。 」我開了大門，有位陌生男人站著，樣子斯文青俊，年約三十歲，笑容可掬。

「 你找誰？ 」

「 先生，您好，我叫ＶＩＣＴＯＲ，是推鎖員，有樣貨品想介紹給您用的。」

正想拒絕之際，他從公事包拿出一根仿真度相當高的自慰器，而且是歐美猛男的尺碼，有２２厘米那麼長，粗度也有４至５厘米。想不到有人會上門推銷成人用品，即時令我尷尬起來。

「 先生，這根自慰器比市面上買到的不同，您試試摸摸，這是用高科技製造，跟真正陰莖上皮膚的質感完全無分別。」他遞給我，不接好像有點不禮貌，又很好奇想摸一下。於是接過來撫摸，啊！真的跟老二上的皮膚一模一樣，而且暖暖的，龜頭上的光澤紋理，完全照足真實來製造。

「 請試試湊近鼻孔嗅嗅。」我拿近鼻子嗅著，哎呀！連那種龜頭上的騷味也能造到，頓然一股熱流湧向下體，老二勃了起來，頂著內褲。

「 請您抓抓那個陰囊部份，它能感應到刺激而收縮或鬆弛，睪丸部份也造得相當仔細，手感十分試試搓揉一下。」那陰囊在手上真的會慢慢縮起，真神奇。我的老二頂得十分辛苦了，不如立即買下來試試。

「 那麼賣多少錢？」估計也不會便宜。

「 請不用心急，我還沒有介紹完畢。可以讓我進來示範嗎？」

示範？有甚麼地方需要示範？好奇心的驅使下，我開了鐵閘門讓他進來。我們走到客廳坐下，他打開公事包，裡面還有一些成人情趣用品。他拿了一瓶載滿白色液體的瓶子，和一支潤滑油。

「 麻煩您給我。」他指著那根自慰器。他拿了指著它的尾部。

「 這裡可以順針方向扭開來，就是這樣。看，這邊放一顆２Ａ電池，而這裡有個小孔。」他拿起那瓶子打開了瓶蓋，瓶口像針管的，將那針管狀的瓶口插進自慰器那小孔中，「 您只須用小力按按瓶身，那些乳液便會注進自慰器裡。又請您試試嗅嗅這些乳液。」他將那瓶子從小孔抽了出來，在我手掌心注了少少。我放在鼻子前嗅，哇，是新鮮的精液味道。

「 這些跟真正精液的味道是完全一樣，您舔一下便會發覺，無論色，香，味和密度跟一般精液並沒分別，是使用多種食物的汁液再加工調製而成，不含防腐劑，不但能夠食用，還對身體有益的。」伸出舌頭舔了一口試試，啊！根本就是精液的味道，神乎奇技！他將有吸盤的自慰器尾部扭上，再在它的莖身上塗了些潤滑油。

「 現在我示範一下。」他解開褲頭皮帶，拉下褲子拉鍊，你……想幹甚麼？他的西褲滑下到腳跟上，呀！ ！不是嗎？連雪白的內褲也褪了下去。

軟垂有少許包莖的雞巴陳現眼前，這根真人肉腸跟仿做的同樣吸引我，誘惑我。不行了，老二快頂穿內褲啊。他轉身抹了些潤滑油在屁眼，噢，那軟軟的屁眼皺起來，經過潤滑油的潤飾，真想舔吮那菊花蕊蜜。他拿起那根自慰器塞進自己的屁眼裡去。

「 啊………」他昂首呻吟。

「 只要…這自慰器。插。進屁。眼裡，它會感應。到體溫…而蠕動。起來。」看到自慰器的尾部在震動，「 如果…將它吸在…光滑的表面…如牆壁，桌面，坐椅上，它便…會上下…插動。」他拔了自慰器出來，將它吸在牆上，那根自慰器真的前後插動，他彎下腰身體背向著自慰器後退那根插動的自慰器直沒進他體內，「 嗯……十…分…爽。先生，要…試試。嗎？」

我毫不考慮便站起來，脫下那條礙頭礙莖的內外褲，老二１８０度朝天彈起來，硬直的貼著小腹，他看見我進入興奮狀態的老二，便向前踏去脫離那自慰器，他想走開讓我來試，可是我走上前拉著他，按下他身體，「 先生…」不等他說完便將急色鬼老二插進他的屁眼去。

「 啊…您…啊…」我拉著他一齊向後退，直至那自慰器插進我的肉洞去。

「 哇…好…好……好…啊…」那根自慰器在插動我的緊穴，而我又在抽插著那推銷員，雙重享受一樂也。我的手還套弄他那不賴的雞巴，抽插了一段時間後，三根陰莖越來越熱，背後的自慰器首先發難，噴射出熱熱的人造精液，接著是那推銷員，他受不了我瘋狂打樁式的套弄，射出幾股乳白的精液，手上也沾了不少，我舔乾手上的精液後，「 啊……………」主角通常最後才出場，發射了近十道的精液，貫滿他的屁眼肉穴a

「 我…還…未介…紹…完它射了…後，若不用…嘴巴清…潔乾淨它，下次…它…不會…動的。」 他不停地喘息。

「 我…本來…想買它，現…在…我想…連你…也一併買下…來，我們…進房繼續…快活。」我拉著他走進房間，關上門。

餵，主人，你忘了替我清潔啊！
